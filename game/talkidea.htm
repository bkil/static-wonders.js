Introduction

Socialize with friends by speaking about more in-depth topics than small talk. You can keep track of which questions you had already discussed in each social circle by assigning each session to a group. You can create a new group or select an existing one. The same question can be discussed in different groups. The default survey set is Copyright (c) 2016 Lukas Subel, Panphico, Arndt Adler and can be distributed under the MIT license, originally from:

  • Ask me anything meaningful - meaningful questions for a good conversation

You can install a web browser that supports at least JavaScript 1.0 (1995) to run this application or you may consider implementing a GemiWeb0 browser with JavaScript0 yourself according to the following specification:

  • bkil.gitlab.io/gemiweb0
