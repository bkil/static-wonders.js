Usage

Move all disks from the first rod to the second rod to solve this puzzle game. You may only move the last (top) disk on each rod. You may only place a smaller disk on top of (following) a larger disk.

Requirements

You can install a web browser that supports at least JavaScript 1.0 (1995) to use this tool or you may consider implementing a GemiWeb0 browser with JavaScript0 yourself according to the following specification:
