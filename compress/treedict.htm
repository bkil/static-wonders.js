Introduction

Compute a storage-efficient radix tree representation from a given ordered list of dictionary words. The suffix graph is serialized to regexp syntax. Also supports reversing the transformation. The end result will be even smaller when compressed with common general purpose algorithms such as gzip. This will reduce the package size of software wishing to bundle a lexicon.
